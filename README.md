# Welcome to KeyBoard Game

## Game concept 
Practice key placement and typeing

## Run the game
clone the repo
```
git clone https://github.com/Theacebutler/keyboard_game.git
cd keyboard_game
```
Run the game
```
python key_game
```
Exit with ctrl+C or ctrl+D
